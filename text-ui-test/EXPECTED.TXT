💰 Welcome to RLAD - Record Losses And Debt!
Manage your finances from the command line - no spreadsheet headaches.

📌 Quick Start Examples:
  add credit 3000 2026-03-01 salary "March salary"     # Add income
  add debit 15.50 2026-03-05 food "Lunch"              # Add expense
  list                                                 # View all
  list type:debit                                      # Show expenses
  budget set 2026-03 1 500                             # Set food budget
Type 'help' for full documentation or 'help <command>' for details.

📋 Available Commands:
  add       - Record income (credit) or expense (debit)
  delete    - Remove a transaction by ID
  modify    - Edit an existing transaction
  list      - View transactions (with optional filters)
  sort      - Set global sort order (amount/date)
  summarize - Show spending summary and breakdown
  budget    - Manage monthly budgets by category
  export    - Save transactions to CSV file
  import    - Load transactions from CSV file
  search    - Find transactions by keyword
  clear     - Delete ALL transactions (with confirmation)
  help      - Show this help or command-specific help
  exit      - Quit the application

💡 Quick Tips:
  • Use 'list type:debit' to see only expenses
  • Use 'list cat:food' to filter by category
  • Use 'list from:2026-03-01 to:2026-03-31' for date ranges
  • Use quotes for multi-word descriptions: "chicken rice"
